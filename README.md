# TaskFlow-Task-Management-System

## Branch Purpose

The `feature/backend-api` branch handles **server-side logic and API development** for TaskFlow.

It provides data services required by the frontend UI.


## Scope

- Authentication APIs
- Task CRUD APIs
- Workflow management logic
- User & role handling
- Data validation  


## Tech Stack

- Backend Framework
- REST APIs
- JSON data handling


## Workflow

- API features developed here
- Tested independently
- Merged into `develop`


## Notes
