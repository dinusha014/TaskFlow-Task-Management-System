# TaskFlow-Task-Management-System

## Branch Purpose

The `feature/frontend-ui` branch is dedicated to **designing and developing the user interface** of TaskFlow.

This branch focuses purely on **visual design and user experience**, without backend integration.

## Scope

- Landing page
- Login & Signup UI
- Dashboard UI
- Task, Workflow & Reports UI
- Responsive design  


## Tech Stack

- HTML
- CSS
- JavaScript
